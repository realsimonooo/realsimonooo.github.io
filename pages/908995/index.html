Web API

  • Web APIs 包含DOM BOM，是JS的应用

    与页面交互功能有关；

# API

  • API（Application Programming Interface,应用程序编程接口）是一些预先定义的函数，目的是提供应用程序与开发人员基于某软件或硬件得以访问一组例程的能力，而又无需访问源码，或理解内部工作机制的细节。

    API 是给程序员提供的一种工具，以便能更轻松的实现想要完成的功能。

# Web APIs

是浏览器提供的一套操作浏览器功能和页面元素的API (BOM/DOM)；

​ 例，常用的警示框alert()是 API；

  • API 提供接口，帮助实现某种功能，不用考虑内部实现。
  • Web API 主要是针对于浏览器提供的接口，主要针对于浏览器做交互效果。
  • Web API 一般都有输入和输出（函数的传参和返回值），Web API 很多都是方法（函数）学习Web API 可以结合前面学习内置对象方法的思路学习；
