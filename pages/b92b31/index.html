作用域

# 作用域

限定代码中所用到的名字的可用性代码范围——作用域；

  • 作用域提高了逻辑的局部性，增强程序的可靠性，减少了名字冲突

JS 中的作用域有两种：

  • 全局作用域——作用于所用代码的执行的区域（script标签内或者独立的JS文件）；
  • 局部作用域 / 函数作用域 —— 在函数内的代码环境；

在JS中没有块级作用域；

// example
if(...){
	var num = 1;
}
console.log(num); // 可以获取num
