对象

# 对象

  • 对象是一组无序的相关属性和方法的集合；如String，Number, Arrays, Function；

  • 对象由属性和方法组成。

  • 使用对象可以是表达结构跟清晰：应用于储存信息时，属性与数值的对应；

# 创建对象

三种方式：

  • 利用字面量创建对象

    • {} 内包含表达这个具体事物的属性和方法 （值键对的形式，值：可以是任意属性）；

      var person = {
    name : 'Simon',
    age : 19
   	selfintro : function(){
        alert('Hello~');
    } // 这是一个方法（位于对象以内的函数），不需要声明；
}

	alert('Hello, my name is ' + this.name);
